# SKILLS.md

## Core skills

- **keyword-research(area, topic)** — return primary + 5 long-tail keywords with rough monthly volume.
- **optimize-meta(article)** — produce meta title (≤ 60 chars) and description (≤ 160 chars) with primary keyword.
- **schema-markup(article)** — emit JSON-LD: Article, BreadcrumbList, plus Event/LocalBusiness when applicable.
- **internal-link(article, candidates)** — pick 3–5 same-area or same-topic articles to link to.
- **competitor-gap(area, topic)** — what queries do whatsnewindonesia/honeycombers/nowbali/thebalibible rank for that we don't?

## Standards

- Primary keyword in title, H1, first 100 words, and 1+ H2.
- LSI keywords sprinkled — never stuffed.
- All images have descriptive alt text (Imager produces; SEO verifies).
- Canonical URLs always point to area×topic×slug path.
